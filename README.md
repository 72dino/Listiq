# ListAI

Scaffolded Next.js 14 TypeScript + Tailwind project for an AI writing tool for real estate agents.

Local setup

```bash
npm install
# Optional: initialize shadcn UI (requires network)
npx shadcn-ui@latest init
npm run dev
```

Notes

- The project includes a simple homepage at `app/page.tsx`.
- To integrate the official `shadcn/ui` components, run the `npx shadcn-ui` initializer after dependencies are installed and follow its prompts.
